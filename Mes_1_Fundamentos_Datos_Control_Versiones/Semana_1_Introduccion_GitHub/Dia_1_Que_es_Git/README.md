## Reflexión: ¿Cómo ayuda Git en proyectos colaborativos de análisis de datos?

Git permite:
- **Control de versiones:** Seguimiento de cambios en notebooks, scripts, y datos.
- **Colaboración:** Varios contribuyentes pueden aportar simultáneamente, con historial y atribuciones claras.
- **Organización:** Facilita la estructuración del proyecto y la integración de documentación.
- **Automatización:** Integrar scripts y workflows que se actualizan y manejan mediante commits.
- **Revisión y aprendizaje:** Analizar el historial de commits y aprender de aportes previos y buenas prácticas.
